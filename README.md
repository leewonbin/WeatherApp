# Project Overview (프로젝트 개요)
- 프로젝트 이름: WeatherApp [Mini Project]
- 프로젝트 설명: 세계 도시의 현재 날씨 조회 앱

# Key Features (주요 기능)
- **날씨 정보 조회**:
  - 도시명을 입력하면 해당 도시의 날씨 데이터를 가져옵니다.

- **도시 정보 관리**:
  - 도시를 추가하고 삭제 할 수 있습니다.

- **현재 위치 날씨 조회**:
  - 현재 위치의 날씨 정보를 조회 할 수 있습니다.
